# Bookstore Web-App

Ein Übungsprojekt im Rahmen des Moduls 7 der Weiterbildung zum Webentwickler. Ziel des Projekts war die Anwendung von Vanilla JavaScript zur DOM-Manipulation, Datenhaltung über LocalStorage und die Implementierung einer interaktiven Benutzeroberfläche.

## Features

- **Dynamisches Rendering:** Bücher werden aus einem JSON-Array geladen.
- **Interaktive UI:** - Like-System für jedes Buch.
    - Kommentar-Funktion (Hinzufügen & Löschen) mit Echtzeit-Aktualisierung.
    - Favoriten-System inklusive Filter-Funktion.
- **Datenpersistenz:** Speicherung aller Benutzerinteraktionen im Browser-LocalStorage.
- **Enter-Support:** Einfache Benutzerführung durch Absenden via Enter-Taste.

## Tech Stack

- HTML5
- CSS3 (Flexbox, Neon-Glow-Effekte)
- Vanilla JavaScript (ES6+ Module)

## Über das Projekt

Dieses Projekt entstand in meiner sechsten Woche der Weiterbildung. Der Fokus lag darauf, funktionale Logik (State Management) mit einer sauberen und modernen Benutzeroberfläche zu verknüpfen.
